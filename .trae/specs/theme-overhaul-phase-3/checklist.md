# Theme Overhaul Phase 3 Checklist

- [x] Visual Textures
    - [x] "Glass" theme has subtle noise grain.
    - [x] "Cyberpunk" theme has moving scanlines.
    - [x] "Brutal" theme has halftone or grid dots.

- [x] Window Styling
    - [x] Window controls match theme (Dots for Glass, Squares for Brutal).
    - [x] Window entry animation feels distinct per theme.

- [x] Micro-interactions
    - [x] Buttons in Brutal theme hard-shift when clicked.
    - [x] Cards in Glass theme have a shimmer/glow on hover.
    - [x] Text in Cyberpunk theme glitches occasionally or on hover.

- [x] Performance
    - [x] Animations do not cause significant layout thrashing.
    - [x] Texture overlays use `pointer-events: none` and do not block interaction.
